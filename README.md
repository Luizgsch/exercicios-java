# Exercícios em Java

Repositório criado para registrar minha evolução na linguagem Java.  
Aqui estarão exercícios de lógica, estruturas condicionais, loops, vetores e demais fundamentos.

## Objetivo
Praticar sintaxe, raciocínio lógico e boas práticas na programação Java.

## Conteúdo
- Estruturas condicionais (`if`, `else`, `switch`)
- Estruturas de repetição (`for`, `while`, `do-while`)
- Operadores aritméticos e lógicos
- Entrada e saída de dados
- Exercícios diversos de lógica

## Como executar
1. Certifique-se de ter o **Java** instalado.
2. Compile o arquivo:
```

javac Main.java

```
3. Execute:
```

java Main

```
